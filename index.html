Example 10: The CSS Box Model

Box 1: Border

This box has a thick 5px solid dark red BORDER around it. The border is the line that wraps around the content.

Box 2: Padding

This box has 30px of PADDING. Padding creates space INSIDE the box between the text and the border. Notice how much breathing room the text has!

Box 3: Margin

This box has 40px of MARGIN. Margin creates space OUTSIDE the box, pushing other elements away. Notice the large gap between this box and the others!

Example 11: Float, Clear, and Clearfix

Experiment 1: Float Left

This image is floated to the LEFT. Notice how the text wraps around it! Float removes the element from normal document flow and pushes it to the specified side (left or right). The text flows around the floated image naturally.

Float is commonly used for wrapping text around images or creating multi-column layouts.

Experiment 2: Float Right

This image is floated to the RIGHT. The text wraps around it on the left side.

This paragraph has "clear: right" which means it will NOT wrap around right-floated elements. It moves below the floated image instead.

Experiment 3: Clearfix

Both images are floated left. The parent div uses clearfix to properly contain the floated images. Without clearfix, the parent div might collapse and not show its background color around the floated elements!

Example 12: CSS Position Properties

1. Position: Static (Default)

This image has position: static, which is the DEFAULT. Elements with static position follow the normal document flow. Top, bottom, left, right properties have NO effect on static elements.

2. Position: Relative

Normal position reference:

Same image with relative positioning (moved 30px left and 20px down from its normal spot):

Position: relative moves the element RELATIVE to where it would normally be. The space it originally occupied is still reserved!

3. Position: Absolute

This container has position: relative (the parent). The bird image inside has position: absolute.

Position: absolute removes the element from normal flow and positions it relative to its nearest POSITIONED ancestor (parent with position: relative, absolute, or fixed). Here it's positioned 50px from top and 20px from right of this blue container.

4. Position: Fixed

Look at the bottom-right corner of your browser window! There's a small bird image with position: fixed.

Position: fixed removes the element from normal flow and positions it relative to the BROWSER WINDOW. It stays in the same place even when you scroll! The bird will always stay in the bottom-right corner.
